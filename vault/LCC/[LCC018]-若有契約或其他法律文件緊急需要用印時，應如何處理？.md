# 若有契約或其他法律文件緊急需要用印時，應如何處理？

## Answer

*原則：
1.預知緊急需求時，應以「簽呈」申請，說明例外項目與完成時限。簽呈同意後，依正常程序申請，HQ將彈性處理。
2.若遇無法預知的緊急需求，可先提交「法律文件審核申請單(含用印)」，並由申請單位主管與承辦單位主管協商優先處理。如無共識，則升級至雙方共同主管決定。
3.惟申請前，應先判斷契約性質，依對應流程提出用印需求。
*步驟：
1.申請人-提交申請：
 (1)判斷契約性質後，依適用流程申請：
      a.一般契約：提交BPM「法律文件審核申請單(含用印)」，申請單位主管簽核後，依步驟 2. 向LCC主管提出審查。
      b.採購契約：確認已完成IPC「採購契約審查申請單」審查後，依步驟 2. 向LCC主管提出審查。
      c.機密契約：一律採紙本用印，請彌封文件後直接交由LCC同仁安排用印，無需依一般契約流程辦理。
2.LCC主管-審查申請：
  (1)同意則緊急處理。
  (2)若不同意，申請單位主管升級至CEOT決定。
