# 如何辦理新商標註冊？

## Answer

*原則：​
若有申請商標之需求，請於評估需求後，透過「簽呈」向LCC提出申請。​
*步驟：​
1.如何評估申請新商標的必要性及周延性：​
  (1)必要性：請評估商機發展速度，決定欲申請的商標使用區域。因商標註冊及維護成本高，特別是國外申請費用昂貴，建議先在台灣註冊以因應商業發展（可參照下附連結規劃商標註冊預算）。​
  (2)周延性：若要申請註冊國外商標，請先評估要申請哪個國家。若太早申請註冊，卻在當地都沒有使用，可能導致無效益或被撤銷（相關規定需依照註冊國的法規，若有需要進一步了解可另洽詢LCC）。​
2.LCC確認：​
  (1)完成前述評估後，請備妥以下資料，請透過「簽呈」加簽LCC同仁提出申請（參照FAQ[PDO006]）：​
      a.商標設計圖​
      b.商標申請名稱​
      c.商標申請類別（至少要能說明日後商標會用於哪些服務或商業活動。）​
      d.商標使用區域（請說明商標預計使用的區域或國家。）​
  (2) LCC確認上述資訊後，將檢索商標資料庫是否有近似商標並提供建議。​
3. LCC委外辦理：​
LCC彙整資訊後，委託商標事務所辦理。
