# 契約製約及用印的例外情況有哪些？

## Answer

*原則：契約製約及用印，應把握五項基本原則，除此外，LCC均認列歸屬例外狀況
  1. 採用電子簽章(由我方用契約檔封包上傳，開啟電子簽章流程)
  2. 由我方製約
  3. 對方先用印
  4. 我方五天內完成用印
  5. 不包含機密契約

*步驟：
認列為例外狀況時，請依以下檢索，查閱FAQ後進行：
1.對方要求使用紙本簽約。參考FAQ[LCC021]廠商/客戶要求使用紙本簽約，應如何處理？
2.對方已先製約用印，要我方用印。參考FAQ[LCC015]收到對方已用印完的紙本契約，應如何申請我方用印作業？
3.需緊急用印契約或其他法律文件。參考FAQ[LCC018]若有契約或其他法律文件緊急需要用印時，應如何處理？
4.機密契約如何用印?  參考FAQ[LCC019]機密契約如何製作紙本契約?(機密契約由承辦單位自行製作)、FAQ[LCC020]機密契約用印流程為何？
