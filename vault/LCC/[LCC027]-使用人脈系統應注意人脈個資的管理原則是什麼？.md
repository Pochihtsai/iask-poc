# 使用人脈系統應注意人脈個資的管理原則是什麼？

## Answer

*定義：
人脈系統啟用後，基於「謀定後動」，單位主管需留意人脈個資的法遵要點，並提醒同仁遵循。
*使用情境：
當有調閱人脈個資需求時，單位主管應依循人脈個資的管理原則，審慎評估需求、留意個資法遵之要項、記錄調閱過程，以強化個資管理。
*提醒：管理人脈個資的原則
1.了解索取人脈個資使用目的
(1)確保提供個資的目的與原始蒐集目的相符，或在法律允許範圍內。
(2)如超出原本目的，需重新取得人脈的同意。
2. 必要性原則
(1)僅提供必要的資料以降低個資外洩風險。
(2)仔細評估申請者需求，避免過度提供。
3. 記錄提供過程
詳細記錄作為稽核或解決爭議的依據。
4. 事先照會人脈
增加透明度，讓人脈了解何人將因何事接觸他。
5. 必要時要求保密承諾
增加一層保護，但不能取代其他安全措施。
