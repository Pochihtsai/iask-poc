# 行動後學習AAR是什麼？如何應用？AAR管理規範須注意哪些事項？

## Answer

*定義：
1.AAR 是 After Action Review 的縮寫，意為「行動後反思」或「事後回顧」。秉持發揮本集團領導公約「學習為本」、「積極創新」，促進持續學習，讓團隊成員在完成行動後能夠進行「行動後學習」，以事實為依據共同進行檢討與回饋。
2.AAR最早是美國陸軍所進行的一項任務後的檢視方法，對美國陸軍來說，使用這種方法的好處是當新的理解被產生時，可以馬上響應到行動上，他們使用AAR解決了許多問題。美國陸軍使用AAR的成功因素包括：結構化、有目地的對談；打破層級的藩籬；能迅速反映至行動中；將學習的成果記錄下來等。
3.AAR是一種知識管理常用的學習與改進方法，目的是從已經完成的行動中獲取經驗教訓，並以此為基礎優化未來的表現。它提供了一個結構化的方式，讓參與者共同回顧和討論所達成的目標、執行過程、問題和解決方案，以及所學到的知識。
4. AAR 對顧問服務業的重要性體現在以下兩個方面：
(1)促進持續創新：AAR 的核心目的是創新，並非反省檢討，且應內化為日常的一部分，持續不斷地縮短理想與現實的差距，以更好地應對複雜多變的顧問服務樣態。
(2)凝聚標準和共識：AAR 可以幫助我們建立統一的服務標準與方法論，確保服務品質始終如一。
*使用情境：
1.AAR通常用於組織、團隊或個人在完成任務、專案或活動之後進行反思和評估。
2.形式：進行AAR的方式可以透過會議，或是發送意見調查收集的方式，針對原設定的目標，引導及彙整參與者提出to Keep、to Improve、Action Item。
3.AAR會議主持人（Facilitator）通常由單位主管或委由專業的引導師擔任，可參考使用「AAR會議簡報格式」引導。
*HQ管理規範：
1.案件結案前須至少完成一場AAR並歸檔；但如果案件實際執行期程超過三個月以上，必須每三個月進行階段性的AAR（因為超過三個月要再回想已較困難，可能遺漏重要lesson）。
2.若屬於一般單位事務，則請依據主管指示進行AAR。
3.請單位主管於AAR完成後一周內提交KMC審核（試行期間先以email提交給KMC FrankC）。
4.AAR彙整請以「AAR彙整提報格式」整理成紀錄。
