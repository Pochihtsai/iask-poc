# 公司是否提供烘碗機供同仁使用？

## Answer

*原則：
公司提供烘碗機予同仁使用，並制定使用規範以維護設備安全及環境衛生。
*提醒：
1.	烘碗機使用前請確保餐具已清潔乾淨，避免食物殘渣殘留，防止異味或污染機器內部。
2.	請將餐具妥善擺放於烘碗機內，避免堆疊過多影響烘乾效果或造成設備損壞。
3.	不耐高溫的餐具（如木製、塑料製品）應避免放入烘碗機，以免變形或損壞。
4.	使用後請隨手關閉烘碗機門，並根據實際需要設定烘碗機運作時間；餐具烘乾後，請儘速取回，勿長期放置烘碗機內。
5.	如發現烘碗機設備異常或有無法運作的情形，請即通知郵務室人員協助通報維修。
6.	每月最後一週之週五下午為烘碗機清潔日，請於當日下午3點前取回個人餐具，逾時未取回之餐具將由清潔人員移置櫥櫃。
