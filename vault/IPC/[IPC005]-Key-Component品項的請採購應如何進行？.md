# Key Component品項的請採購應如何進行？

## Answer

*原則：經由統一採購可確保價格優勢或品質保證之品項即屬於Key Component品項，由IPC統籌處理。品項如下：
1.筆電（含滑鼠、鍵盤）、筆電使用之螢幕、iPhone、 iPad。
2.平面或視覺設計。
3.非VIP為對象的活動舉辦。
4.室內裝修工程。
5.室內弱電(佈纜)相關工程。
*步驟：
1.使用「採購契約審查申請單」向IPC申請，並附上草約（不需議價）或SOW（包含交付項目、時程、驗收條件等必要資訊）。
2.由IPC會同需求單位向Key Component Vendor進行需求brief。
3.由需求單位與IPC共同確認與供應商的草約內容。
4.由IPC進行議價與議約。
