# 文件管理與溝通規範

KMC 主管，涵蓋檔案命名、Email 規範、AAR 學習、ISO 文管。GAC 提供文件範本，TMC 在新人訓含部分規範。

## 流程主軸

```
文件產生
  → 命名規則(KMC005, KMC015)
  → 範本取用(GAC036)
  → 流通: Email 連結(KMC001) + 溝通原則(KMC006)
  → 重要決策溝通: CEOT 通知名單(KMC014, KMC018)
  → 收尾: AAR(KMC008) → 歸檔 → ISO 文管(KMC016)
```
