# 採購與供應商管理

集團採購是**跨四個部門**的協作流程：IPC 主導採購政策、CFC 控管預算與付款、LCC 審核契約條款、GAC 處理小額庶務採購。

## 流程主軸

```
需求 → 預算確認(CFC) → 選商(IPC) → 契約(LCC+IPC)
     → 採購執行 → 驗收(IPC) → 請款(CFC) → 發票/付款(CFC)
```
