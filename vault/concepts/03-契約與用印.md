# 契約與用印

LCC 是法務主管，處理所有契約審核與用印流程。涉及採購時與 IPC 並聯，涉及付款時與 CFC 並聯。

## 流程主軸

```
契約需求
  → 判斷類型: 制式契約(LCC002) / 客製契約 / 機密契約(LCC019)
  → 起草: 範本調閱(LCC003) / 填製要領(LCC006)
  → 審核: 法律文件審核申請單(LCC011) + 送出前檢查(LCC012)
  → 簽署: 紙本 / 線上(LCC017) / 緊急用印(LCC018)
  → 機密契約: 自製(LCC019) + 機密用印流程(LCC020)
```
