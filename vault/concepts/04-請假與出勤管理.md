# 請假與出勤管理

TMC（人資）主管，涵蓋假別申請、出勤打卡、加班、WFH、職代等。

## 流程主軸

```
請假申請
  → 假別判斷: 特休/事假/病假/婚假/喪假/陪產/家庭照顧
  → 申請時機: 預先申請(TMC046) / 緊急請假(TMC049)
  → 職代啟動(TMC006, TMC077)
  → 出勤異常處理(TMC082) / 加班調班(TMC026, TMC028, TMC084)
```
