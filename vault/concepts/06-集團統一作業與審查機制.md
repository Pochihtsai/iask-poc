# 集團統一作業與審查機制

HQ 是集團最高治理層，所有部門遵循 HQ 統一作業與審查標準。每個部門都有自己的「公告管理制度」，由 HQ 統一治理。

## 治理結構

```
HQ（治理層）
├── PDO（流程設計）
├── PCC（公司治理）
├── CFC（財務）
├── GAC（行政）
├── IPC（採購）
├── KMC（知識管理）
├── LCC（法務）
├── PMC（專案管理）
└── TMC（人資）
```
