# 任務與專案管理 lifecycle

PMC 是集團專案管理主管部門，但完整 lifecycle 跨 **PMC（執行）+ HQ（治理）+ CFC（預算）+ LCC（契約）**。

## 流程主軸

```
商機(PMC003)
  → 提案(CEM 4/5) → 任務代號(PMC006)
  → 任務開立(PMC008) → 預算(CFC003)
  → 執行 + Pipeline(PMC005) + Worklist(PMC020)
  → 變更管理(PMC016, PMC032) + RDM(PMC021)
  → 結案(PMC013)
```
