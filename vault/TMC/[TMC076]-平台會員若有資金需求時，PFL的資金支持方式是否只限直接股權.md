# 平台會員若有資金需求時，PFL的資金支持方式是否只限直接股權投資的方式？是否有考慮債權形式提供資金支持，舉例，約定定期收利息、到期還本、或以債作股等方式？

## Answer

*原則：PFL是一家公司，應以獲利為最主要目的來跟平台會員談。先說明釐清，PFL平台輔導會員企業主，並不一定會投資；投資也不一定就邀請對方加入平台。

*使用情境：
1.以實際狀況來看，假設CEOT投資100家企業，其中可能只有5~10家會加入平台當會員，因為投資評估是一件事情，就如創投功能思考投資風險及投資報酬率即可，投入的主要也就是資金，不太需要投入太多其他資源；但平台會員是需要花費PFL團隊及CEOT許多精神去引導、去管理、去協助，也需要PFL平台動用品牌影響力、動用許多人脈去輔導，投入的成本可說是單純投資的100倍也不為過，所以是否同意企業加入平台成為會員的評估是另一件事情，評估方式也跟評估投資不同。
2. 是否邀請一家企業加入平台、是否要投資一家企業、是否會借錢給一家企業，這些都該想成獨立事件，都會有不同的評估。
3. PFL與一家企業的關係，也可能從投資變成會員、也可能從借貸變成投資、也可能就單純借貸而不讓其加入平台等，都要視其評估結果而定。而不論是加入會員、或投資、或借貸，都會由專業的財務規劃及建議最合乎利益的方式，可能是高利息、可能是以債作股等，都有可能。
4. 實際上就有一家企業，PFL平台有借錢給其企業主，收取利息外，也設定一定期間內若沒還，就轉成股票償還，而對方一直想加入平台成為會員，但我們的評估一直無法同意讓它加入平台。
