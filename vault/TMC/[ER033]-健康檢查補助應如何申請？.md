# 健康檢查補助應如何申請？

## Answer

*原則：健康檢查為集團的福利項目，使用福利項目前需取得TMC同意，之後可走一般請款流程。
*步驟：
1.使用「員工健康檢查申請單」向TMC申請，取得同意後可進行檢查。
2.健康檢查得請公假1天，須於健康檢查前3天完成請假手續，並於假單備註欄註明為「健康檢查」。
3.於檢查完畢後的30個工作天內，使用「請款單」進行請款，並附上醫療機構之繳費收據正本。
4.附上「勞工健康保護規則_一般體格、健康檢查項目表」所列項目的檢查資料連結供TMC確認。
5.依年度費用歸屬原則，費用申請作業需於該年度12月5日前完成，逾時無法受理。
