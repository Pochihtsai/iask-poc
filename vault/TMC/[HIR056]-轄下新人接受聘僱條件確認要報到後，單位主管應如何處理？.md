# 轄下新人接受聘僱條件確認要報到後，單位主管應如何處理？

## Answer

（本項僅PFL適用，其他法人請依所屬管理規範作業，如有疑問請洽各法人管理單位。）
*原則： TMSC Hiring必須收到新人簽回的Offer Letter，以及單位主管更新後的五大表給TMC HRM才會進行新人報到作業。
*步驟： 
1.	若用人單位主管與C&B 皆同意雇用，則TMSC Hiring窗口會提供錄取通知書(Offer Letter)給用人單位主管去跟新人談。 
2.	用人單位主管提供錄取通知書給新人，並請新人簽回。
3.	用人單位主管提供新人簽回之錄取通知書給TMSC Hiring窗口。
4.	用人單位主管更新五大表後，提供給TMC HRM窗口。
5.	TMC HRM啟動新人報到作業。
