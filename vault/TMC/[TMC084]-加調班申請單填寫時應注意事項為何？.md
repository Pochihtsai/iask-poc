# 加調班申請單填寫時應注意事項為何？

## Answer

*原則：
1.已連續工作4小時，請先至少休息30分鐘後，再繼續(如工作有連續性或緊急性者，得在工作時間內，另行調配其休息時間)。
   例1：平日17:00下班，加班起始時間為17:30。
   例2：假日8:00-18:00，12:00起須至少休息30分鐘後，再繼續(如工作有連續性或緊急性者，得在工作時間內，另行調配其休息時間)。
2. 調休時間須跟加班時間及時數須一致。
3. 調班日應於加班日之後，但若將造成連續工作7日的情形，申請人的主管請先與TMC聯繫。
4. 主管於簽核時，應於「加調班申請單」中簡要說明加班情形與事由，否則將不予採認。

*步驟：請填寫「加調班申請單」。
